infoporto.companyshowcase Installation
--------------------------------------

To install infoporto.companyshowcase using zc.buildout and the plone.recipe.zope2instance
recipe to manage your project, you can do this:

* Add ``infoporto.companyshowcase`` to the list of eggs to install, e.g.:

    [buildout]
    ...
    eggs =
        ...
        infoporto.companyshowcase
       
* Re-run buildout, e.g. with:

    $ ./bin/buildout
